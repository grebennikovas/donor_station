## Учебный проект для работы с базой данных станции переливания крови
#### Задачи:
- Разработать модель данных для бизнес-процессов на станции переливания крови
- Разработать REST API для взаимодействия с базой данных

#### Технологии:
- PostgreSQL
- Java Spring
